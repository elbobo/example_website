What is CSS?

CSS (Cascading Style Sheets) is the language used to style and layout webpages. It controls colours, fonts, spacing, and positioning of elements.

Key Concepts

  • Selectors - Target which HTML elements to style, such as p, h1, or .classname.
  • Properties - The aspects you want to change, such as color, font-size, or margin.
  • Values - The settings you apply to properties, such as red, 16px, or 20px.

Example Code

Here is a simple CSS rule that makes all paragraphs blue:

p {
    color: blue;
}
